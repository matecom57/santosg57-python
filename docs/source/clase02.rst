Cadenas de Caracteres o Strings

.. code:: Python

   s1 = 'Hola Mundo'

Es lo mismo

.. code:: Python

   s2 = "Hola Mundo"

   type(s2)

Si quieres o necesitas que un string ocupe más de una línea, entonces debes encerrar el texto entre tres comillas simples '''...''' o dobles 
"""...""".

.. code:: Python

   s = '''
   Este string
   ocupa más
   de
   una línea'''

Concatenando string's

.. code:: Python

   hola = 'hola'
   s = hola + '  Mundo'
