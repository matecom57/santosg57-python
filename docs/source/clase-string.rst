Clase String
============

.. code:: Bash

   >>> x = '''
   ...  hola
   ...  como estas,
   ...  yo estoy bien'''
   >>> 
   >>> x
   '\n hola\n como estas,\n yo estoy bien'
   >>> 
   >>> len(x)
   34
   >>> type(x)
   <class 'str'>
   >>> 


   >>> y1 = "1234"
