Clase tupla
===========

La clase tutpla representa un tipo de datos estructurado, cuyos elementos no pueden ser modificados.

Ejemplos:

.. code:: Bash

   x = ()
   type(x)
   
MÃ©todos de la clas tupla:

``'count', 'index'``
 
Ejemplos:

.. code:: Bash

   y = ('a', 'b', 'c', 'a', 'b', 'a', 'c')
   len(y)
   y.count('b')
   y.index('b')
